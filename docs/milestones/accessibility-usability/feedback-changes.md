# Feedback Changes

Usability tests reflected weak points of the application, which affected overall user experience.  
A high number of feedback suggestions was collected, an effort was made to respond to that feedback as fast as possible, which resulted on changed on our user interfaces.

## Settings button
![settings-button](/img/feedback-changes/settings.png)
A new settings button was created, with the objective of reflecting more it's functionality as a clickable

## Settings button
![weather-alerts](/img/feedback-changes/weather.png)
Weather Alerts now have the border color of the warning (for example: Yellow Warning now has a yellow border).

## Sound Toggle
![audio-toggle](/img/feedback-changes/audio.png)  
Sound toggle was changed for better understandibility, it is now a regular toggle instead of a clickable icon. This toggle is also altomatically off when the alert is off.

## New UI
![new-ui](/img/feedback-changes/ui.png)
![ui-while](/img/feedback-changes/ui-white.png)
The new UI of the app has more uniform components.  
On Dark mode non clickable components are gray, while clickables are darker, which aims to easen the indentification of buttons.  
This change is also reflected on the White Mode of the app.
